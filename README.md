# Credit Card Fraud Detection

Built a fraud detection ML model handling imbalanced datasets using SMOTE technique.

## Tools Used
- Python, Scikit-learn, Pandas, Matplotlib

## What I Did
- Handled class imbalance using SMOTE oversampling
- Trained Logistic Regression, Random Forest and XGBoost models
- Evaluated using Precision, Recall, F1-Score and ROC-AUC
- Visualized confusion matrix and feature importance
